#  MyBazar - Java Market Management System

##  Project Overview
MyBazar is a console-based market management system developed in Java.  
The project simulates a basic e-commerce/market automation structure and was built to practice core Java and Object-Oriented Programming (OOP) concepts.

---

##  Features

-  Product management (multiple product categories)
-  Customer and employee management
-  Shopping cart system
-  Order processing
-  Campaign management
-  File-based data loading (users, items, commands)
-  Basic stock and ID tracking system
